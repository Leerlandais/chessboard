Development Journal for the Chessboard

English Version

I'm going to, eventually, make a working Chessboard (for 2 players, obviously I'm not gonna make it with a programmed opponent!). I'm gonna keep a diary of how it's going. All of this is of course visible on Github but I'll include it here, too

  • 16/11/2023 - Introduction to making tables using HTML. I decided to make a chessboard design. Spent some time figuring out how to apply the colouring to the board using the nth-child.
  • 17/11/2023 - Decide I'm going to use this as an ongoing project (outside of the work I'll be doing for the course). Will be intersting to see how it develops. At this time, I've just finished the preformation. I'm getting used to HTML, CSS and using Github.
  • 18/11/2023 - Spent the morning reading up on click events. Between W3C and FreeCodeCamp, I managed to figure out how to get a square to change when clicked. It's a small step but it's a start
  • 19/11/2023 - Updated the Journal page, included a French version - Put piece names on squares - Can't do much more today, need to learn more first.

Version Française

Je vais eventuellement fabriquer un échiquier fonctionnel (pour 2 joueurs, évidemment je ne vais pas le faire avec un adversaire programmé !). Je vais tenir un journal de bord pour suivre l'avancée du projet. Tout cela est bien sûr visible sur Github, mais je vais également l'inclure ici.

  • 16/11/2023 - Introduction à la création de tableaux en utilisant HTML. J'ai décidé de concevoir un échiquier. J'ai passé un certain temps à comprendre comment appliquer la coloration à l'échiquier en utilisant la pseudo-classe nth-child.
  • 17/11/2023 - J'ai décidé d'utiliser ceci comme un projet continu (en dehors du travail que je vais faire pour le cours). Ce sera intéressant de voir comment cela évolue. Pour l'instant, je viens de terminer la préformation. Je commence à m'habituer à HTML, CSS et à utiliser Github.
  • 18/11/2023 - J'ai passé la matinée à me documenter sur les événements de clic. Entre W3C et FreeCodeCamp, j'ai réussi à comprendre comment faire changer un carré lorsqu'il est cliqué. C'est un petit pas, mais c'est un début!
  • 19/11/2023 - Mise à jour de la page du journal, incluant une version française - Placer les noms des pièces sur les cases - Je ne peux pas faire beaucoup plus aujourd'hui, j'ai besoin d'apprendre davantage d'abord.
